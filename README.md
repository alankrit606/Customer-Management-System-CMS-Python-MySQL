# Customer-Management-System-CMS-Python-MySQL
📌 Customer Management System (CMS) – Python + MySQL

A console-based Customer Management System built using Python and MySQL, designed to perform complete CRUD operations with a secure login system.

🚀 Project Overview

This project allows users to manage customer records efficiently through a command-line interface. It includes features like:

🔐 User Login Authentication
➕ Add Customer
📄 View All Customers
🔍 Search Customer by ID
✏️ Update Customer Details
❌ Delete Customer
🛠️ Tech Stack
Language: Python 🐍
Database: MySQL 🗄️
Library: pymysql
Concepts Used:
Functions
Exception Handling (try-except-finally)
SQL Queries (CRUD)
Database Connectivity
